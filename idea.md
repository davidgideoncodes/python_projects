# Project Ideas

## 🏫 School Student Management System

### What it does
- Teachers log in and record student grades per subject
- Students log in with ID and check results
- Result portal only opens when school releases it
- Admin controls everything

### Student ID Format
FGC-2609001
- FGC = school code
- 26 = enrollment year
- 09 = enrollment month
- 001 = auto number

### User Roles
- Admin — controls everything
- Teacher — records grades per subject
- Student — views results with ID
